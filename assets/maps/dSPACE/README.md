# dSPACE Maps for Scenic

This directory contains OpenDRIVE and dSPACE RoadNetwork assets for the Laguna Seca racing circuit.

## Files

| File | Purpose |
|------|--------|
| **LagunaSeca.xodr** | Main OpenDRIVE map used by the racing domain, visualization, and dSPACE simulator (default map). |
| **Laguna_Seca.rd** | dSPACE RoadNetwork (.rd) file; used for coordinate transformation and (s,t) projection when both XODR and RD are present. |
| **Laguna_Seca_transform.json** | Cached XODR→RD coordinate transform (generated by the dSPACE geometry pipeline if missing). |
| **LagunaSeca_MainTrack_FromTTL.xodr** | Main-track-only XODR produced by `create_new_ttl/build_main_track_xodr.py`; used by the TTL tooling (e.g. interactive map visualizer). |
| **README.md** | This file. |

## Usage

- **Racing / visualization:** Use `LagunaSeca.xodr` as the map (e.g. `--map assets/maps/dSPACE/LagunaSeca.xodr` or default in the segment visualizer).
- **dSPACE simulator:** The dSPACE model and geometry pipeline use `LagunaSeca.xodr` and, when available, `Laguna_Seca.rd` plus `Laguna_Seca_transform.json` for accurate placement and readback.

Conversion from RD to XODR: `src/scenic/simulators/dspace/converters/rd_to_xodr.py`.
